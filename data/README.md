# Data

This directory contains the data files used in the study.

## Structure

- `raw/` - Raw experimental data files
- `processed/` - Preprocessed data ready for modeling

## Data Description

Add your data files here. For each dataset, please provide:

1. **Data format**: Description of file format (CSV, HDF5, Excel, etc.)
2. **Variables**: List of columns/variables and their meanings
3. **Units**: Measurement units for each variable
4. **Source**: Where the data came from (experiments, simulations, etc.)
5. **Processing**: Any preprocessing steps applied

## Guidelines

- For large data files (>100 MB), consider using Git LFS or hosting on external platforms (Zenodo, figshare, etc.) and provide download links in this README
- Include metadata files describing the experimental conditions
- Document any data transformations or normalization applied
