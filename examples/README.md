# Examples

This directory contains example scripts and Jupyter notebooks demonstrating how to use the code.

## Suggested Examples

1. **Data preprocessing example** - How to load and preprocess the data
2. **Model training** - How to train the hybrid model
3. **Prediction and evaluation** - How to make predictions and evaluate results
4. **Visualization** - How to reproduce figures from the paper
5. **Sensitivity analysis** - Parameter sensitivity studies
6. **Metabolic phase detection** - Detecting metabolic phase transitions

## Format

Examples can be provided as:
- Jupyter notebooks (`.ipynb`) - Interactive, great for exploration
- Python scripts (`.py`) - Standalone scripts with clear documentation
- Both formats covering the same examples
