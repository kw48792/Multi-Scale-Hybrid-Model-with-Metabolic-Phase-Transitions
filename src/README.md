```
src/

├── data_preprocessing.py       # Data loading and preprocessing

├── data_driven_model.py        # Machine learning/data-driven components
├── hybrid_model.py             # Hybrid model combining both approaches
├── metabolic_transitions.py    # Metabolic phase transition detection
├── optimization.py             # Model optimization routines
├── utils.py                    # Utility functions
└── visualization.py            # Plotting and visualization functions
```
